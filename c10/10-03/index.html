    1. 安装

    1.0 rpm安装

    首先通过  rpm -q <关键字> 可以查询到rpm包的名字
然后 调用 rpm -e <包的名字> 删除特定rpm包
如果遇到依赖，无法删除，使用 rpm -e --nodeps <包的名字> 不检查依赖，直接删除rpm包
如果恰好有多个包叫同样的名字，使用 rpm -e --allmatches --nodeps <包的名字> 删除所有相同名字的包， 并忽略依赖

    1.1 离线安装java

    wget --no-check-certificate --no-cookie --header "Cookie: oraclelicense=accept-securebackup-cookie;" http://download.oracle.com/otn-pub/java/jdk/8u73-b02/jdk-8u73-linux-x64.rpm
rpm -ivh 8u73-b02/jdk-8u73-linux-x64.rpm
java -version

    1.2 离线安装jenkins

    wget http://pkg.jenkins-ci.org/redhat-stable/jenkins-2.7.4-1.1.noarch.rpm
rpm -ivh jenkins-2.7.4-1.1.noarch.rpm
sudo service jenkins restart

    1.3 修改目录的相应权限，修改root为当前用户。

    配置目录： /etc/sysconfig/jenkins
程序： /var/lib/jenkins
cache：/var/cache/jenkins 
日志：/var/log/jenkins

    2. 常用配置

    2.1 全局变量 (非常重要)

    jenkins -> 管理Jenkins -> 系统设置 -> 全局属性 -> Environment variables
    序号 变量 备注
    1 jenkins_email_to_list 邮件收件人列表，例：x@xx.com 或者 x@xx.com,y@xx.com
    2 jenkins_email_cc_list 邮件抄送人列表，例：x@xx.com 或者 x@xx.com,y@xx.com
    3 jenkins_project_run_test 是否执行代码中的单元测试，例：只有设置为1，才会执行单元测试（默认为0）
    4 jenkins_git_branch 拉取项目的git分支，例：dev，建议release
    5 jenkins_input_approve_list input中可确认通过的用户列表，例：user1，user2
