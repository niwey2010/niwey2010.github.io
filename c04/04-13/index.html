    Jre 1.8

    • 安装
    安装页面：http://www.oracle.com/technetwork/java/javase/downloads/index-jsp-138363.html
安装包路径：安装页面有jdk-8u111-linux-x64.tar.gz下载地址
安装命令，解压到 /usr/local/java ：
tar -xvzf jdk-8u111-linux-x64.tar.gz
    • 修改环境变量文件， vim /etc/profile :
    export JAVA_HOME=/usr/local/java
export CLASSPATH=.:$JAVA_HOME/jre/lib/rt.jar:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
export PATH=$PATH:$JAVA_HOME/bin
    • 环境变量生效
    source /etc/profile
    • 修改/usr/bin的软链接
    sudo rm -rf /usr/bin/java
sudo ln -s /usr/local/java/bin/java  /usr/bin/java

    python 2.7.x

    • 安装相关的依赖
    yum group list
yum groupinstall "Development tools" -y
yum install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel -y
    • 安装详情
    1. 安装页面：https://www.python.org/downloads/release/python-2713/

2. 解压编译安装
tar -xvzf Python-2.7.13.tgz
cd Python-2.7.13
./configure --prefix=/usr/local --enable-shared
make
make altinstall
echo "/usr/local/lib" > /etc/ld.so.conf.d/pyton.conf
ldconfig

3. 修改/usr/bin的软链接
sudo rm -rf /usr/bin/python
sudo ln -s /usr/local/bin/python2.7 /usr/bin/python

4. 测试是否修改成功
python -V

3. 解压：
unzip setuptools-32.3.1.zip
cd setuptools-32.3.1
python setup.py install

    Supervisord

    • 离线安装 meld3-1.0.1（依赖库）
    安装页面：https://pypi.python.org/pypi/meld3
安装包路径：页面有meld3-1.0.1.tar.gz下载地址
安装命令：
tar -xvzf meld3-1.0.1.tar.gz

    • 安装
    安装页面：https://pypi.python.org/pypi/supervisor
安装包路径：安装页面有supervisor-3.3.1.tar.gz下载地址
安装命令，解压并安装：
tar -xvzf supervisor-3.3.1.tar.gz
cd supervisor-3.3.1
python setup.py install
    • 创建配置
    echo_supervisord_conf > /etc/supervisord.conf
    • 修改配置
    # 修改节点(chown修改为当前用户):
[unix_http_server]
chmod=0700                 ; socket file mode (default 0700)
chown=user:user       ; socket file uid:gid owner

# 修改节点（web管理页面):
[inet_http_server]         ; inet (TCP) server disabled by default
port=0.0.0.0:7800        ; (ip_address:port specifier, *:port for all iface)
username=user              ; (default is no username (open server))
password=123               ; (default is no password (open server))

# 修改节点（添加项目配置路径，支持*通配符）：
[include]
files = /etc/supervisor/conf/*.conf
    • 项目配置 app-*.conf(范例)
    [program:redis-7000]
command=redis-server  /home/user/cluster-test/7000/redis.conf    ; supervisor启动命令
directory=/home/user/cluster-test/7000/                                                ; 项目的文件夹路径
startsecs=0                                                                             ; 启动时间
stopwaitsecs=0                                                                          ; 终止等待时间
autostart=true                                                                         ; 是否自动启动
autorestart=true                                                                       ; 是否自动重启
stdout_logfile=/home/user/conf/log/redis-7000.log                       ; log 日志
stderr_logfile=/home/user/conf/log/redis-7000.err
    • 设置为开机启动
    修改 /etc/rc.local,末尾添加一行：
/usr/local/bin/supervisord

    • openssl-1.1.0c(依赖)

tar -xvzf openssl-1.1.0c.tar.gz

cd nginx

    confd v0.11.0

    • 下载
    wget https://github.com/kelseyhightower/confd/releases/download/v0.11.0/confd-0.11.0-linux-amd64
    • 安装
    # 复制到bin目录
sudo cp confd-0.11.0-linux-amd64 /usr/local/bin/confd

# 修改文件权限
sudo chmod 755 /usr/local/bin/confd

# 修改文件所有
sudo chown user:user /usr/local/bin/confd

 /usr/local/bin/confd -h
    • 查看版本
    /usr/local/bin/confd -version

    新增端口
#sudo /sbin/iptables -I INPUT -p tcp --dport 80 -j ACCEPT
#sudo /sbin/iptables -I INPUT -p tcp --dport 22 -j ACCEPT
